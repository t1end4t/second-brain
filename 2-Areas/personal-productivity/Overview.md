# Personal Productivity

## Why This Area Exists

This area holds the ongoing system for thinking, planning, reviewing, and deciding what to work on.

It is not a project because there is no finish line. It supports the whole vault.

## Current Direction

Use the vault as a daily and weekly operating system:

- Review what was done yesterday.
- On Monday, review what was done last week.
- Decide urgent, needed, important, waiting, and later work.
- Update project and area task files so planning does not depend on memory.
- Close the day by recording done work, blockers, and tomorrow's next action.

## Key Questions

- What should I work on today?
- What did I finish yesterday?
- What changed since the last review?
- What is urgent, important, waiting, or stale?
- Which project or area needs attention next?
