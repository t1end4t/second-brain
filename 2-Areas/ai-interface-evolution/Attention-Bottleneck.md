# Human Attention Bottleneck

**Status:** Active
**Created:** 2026-05-13

---

## Core Claim

As models become stronger and context windows grow, the limiting factor shifts from model token budget to human attention.

The optimization target changes from:

```text
minimize tokens
```

to:

```text
maximize human understanding per minute
```

## Why HTML Helps

HTML can compress review work through:

- progressive disclosure
- visual hierarchy
- filtering and sorting
- tabs and collapsible sections
- diagrams and spatial layout
- embedded controls
- state compression

## Practical Implication

A concise Markdown spec nobody reads can be worse than a larger interactive artifact people actually explore.

But this only holds when the artifact reduces review fatigue, navigation cost, or comprehension time.
