AI-Generated Paper Content Detection Agent

A lightweight coding-agent helper that reviews manuscript text, gathers passage-level evidence, and produces a careful report. It does not accuse, reject, or decide. It helps a human editor inspect.

1. Input

Academic paper content.

  • Markdown/plain text first
  • PDF later if needed
  • Optional references section
→

2. Segment

Split into inspectable units.

  • Sections
  • Paragraphs
  • Sentences
  • Offsets/spans
→

3. Analyze

Run small evidence collectors.

  • Stylometry
  • LLM rubric
  • Citation sanity

4. Aggregate

Combine signals conservatively.

  • No single-score verdict
  • Require evidence spans
  • Apply ESL caveats
→

5. Report

Human-review artifact.

  • Flagged passages
  • Reasons + quotes
  • Recommended questions
→

6. Human Review

Editor decides next steps.

  • Ask author for clarification
  • Check draft history
  • Never auto-reject

Agent modules

Segmenter

Creates stable passage IDs and offsets so every finding maps back to exact text.

Stylometric heuristics

Checks burstiness, transition uniformity, lexical tells, hedging, paragraph regularity, specificity deficit.

LLM judge

Optional rubric pass. Must cite verbatim evidence and include a dissent reason.

Citation sanity

Basic checks: missing reference section, suspiciously generic citations, malformed references, citation/reference mismatch.

ESL guardrail

Reduces confidence when signals may come from L2 writing, translation, formal style, or grammar simplification.

Reporter

Emits markdown or JSON with passages, evidence, caveats, confidence band, and suggested human questions.

Core rule

Evidence packet, not verdict

The agent should say: “This passage has low sentence-length variance, generic transitions, and no concrete methodological details.”

It should not say: “This paper is AI-written.”

Do

  • Quote exact spans
  • Explain features
  • Show uncertainty
  • Recommend review questions

Don’t

  • Auto-reject
  • Use one detector as truth
  • Penalize ESL style
  • Output accusation language
