# Area Instructions

This folder is an ongoing area, not a finite project.

## Read First

- Read `Overview.md` before changing direction.
- Read `TODO.md` before planning or prioritizing.
- Read `Review.md` when helping with daily or weekly review.

## Write Rules

- Keep raw context in the most relevant area note.
- Put concrete actions in `TODO.md`.
- Use `Waiting` for blocked work.
- Move completed work to `Done` with the date.
- Do not create project structure here unless the user defines a concrete finish line.
