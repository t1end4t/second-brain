# HTML Artifacts

## Definition

HTML artifacts are single-page, self-contained HTML files generated by LLM agents (like Claude Code) as rich, interactive output documents. They replace Markdown for complex deliverables—specs, reports, prototypes, reviews, and custom editors—because they can embed CSS, JavaScript, SVG, and interactivity directly in a portable file that opens in any browser.

## Key Principles

- **Information density**: Tables, SVG diagrams, interactive widgets, and code annotations can all live in one file—far richer than Markdown's flat text.
- **Visual structure**: CSS enables tabs, cards, side-by-side layouts, and responsive design, making long documents navigable.
- **Native sharing**: An HTML file is linkable and viewable in any browser; no render pipeline or viewer plugin required.
- **Two-way interaction**: Sliders, toggles, copy-as-JSON buttons, and live previews let humans manipulate agent output and feed edits back into the loop.
- **Agent-native data ingestion**: Claude Code can read the filesystem, MCP servers, browser context, and git history, then synthesize all of it into a single artifact.

## Common Use Cases

| Category | What the artifact does |
|---|---|
| Specs & planning | Side-by-side options, annotated plans, data-flow mockups |
| Code review | Inline diffs, severity-colored annotations, flowcharts |
| Design & prototyping | Interactive component demos with parameter sliders |
| Reports & explainers | Synthesized multi-source narratives with SVG diagrams |
| Custom editors | Purpose-built forms, drag-and-drop triage boards, prompt tuners with live preview |

## Example Prompts

- *"Generate 6 distinct onboarding approaches as a single HTML grid so I can compare tradeoffs."*
- *"Review this PR in HTML: render the diff, annotate streaming/backpressure logic, color-code severity."*
- *"Prototype a checkout button animation with sliders for easing, duration, and color; add a copy-params button."*
- *"Read our rate limiter code and produce an HTML explainer: token-bucket diagram, annotated snippets, gotchas."*
- *"Build a drag-and-drop ticket triage board in HTML with a 'copy as Markdown' export."*
